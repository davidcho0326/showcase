STAGE 2 · DATA ANALYSIS & STRATEGY

데이터 정제

수집 데이터를 구조화된 분석 데이터로 변환

  • 상품 파싱 (카테고리, 브랜드, 키워드)
  • 핵심 노출 포인트 Top 5 추출
  • 타겟 페르소나 & 니즈 분석
  • 소비자 언어 기반 카피라이팅
👥

인플루언서 분석

최적 크리에이터 매칭 및 심층 분석

  • 티어 분류 (Mega / Macro / Micro)
  • 콘텐츠 강점 분석 (안정성, 일관성)
  • 카테고리 친화도 분석
  • 인게이지먼트 / 조회수 / 참여율
🧠

전략 자동 제안

AI가 완성도 높은 콘텐츠 전략을 생성

  • 소비자 감정 언어 매핑
  • 후킹 전략 (Visual/Audio/Gesture)
  • 콘텐츠 가이드 (조명, 미장센)
  • 시나리오 자동 생성 (15-60초)

최종 검수까지 완료합니다

AI 이미지 생성

각 Step별로 AI 프롬프트를 자동 생성하고, 다중 변형을 만들어 최적의 이미지를 선택합니다. 스타일 설정에 따라 프롬프트가 자동 조정됩니다.

착장 변경 배경 교체 브랜드컷 AI 영상 AI 인플루언서

영상 시나리오 생성

15-60초 분량의 플랫폼 최적화 영상 시나리오를 자동 생성합니다. 각 단계별 시간, 비주얼 설명, 오디오 내레이션, 감정 의도가 포함됩니다.

Shorts/Reels 최적화15초
Hook → Problem → Solution → Proof → CTA

촬영 테크니컬 스펙

조명 세팅(3-point), 미장센, 캡션 구조, 업로드 스펙까지 AI가 기술 가이드를 제공합니다.

최종 검수 & PDF Export

AI가 생성한 시딩 가이드를 담당자가 최종 검토하고 승인합니다. 릴스 시딩 가이드, 캠페인 개요, 시나리오 이미지가 포함됩니다.

착장 변경 RELEASED

현장 촬영 이후에도 착장을 자유롭게 교체할 수 있어 재촬영에 드는 시간과 비용을 절감합니다. 포즈, 조명, 배경은 그대로 유지하면서 의상만 자연스럽게 교체됩니다.

AI 영상 제작 DEVELOPMENT

인물·착장·배경·콘셉트 프롬프트만 입력하면, AI 에이전트가 캐릭터 설계 → 시나리오 기획 → 씬 생성 → 영상 합성 → BGM 작곡 → 품질 검수까지 9단계 파이프라인을 자율 실행하여 30초 광고 브랜드 필름을 완성합니다.

1 사전 준비 (Input)

인물, 착장, 배경 이미지를 입력으로 제공합니다

2 캐릭터 시트 + 씬 생성

3방향 캐릭터 시트 생성 후 시작/종료 프레임 설계

3 영상 합성 + 완성

시작→종료 프레임 기반 모션 생성, AI BGM 작곡 후 30초 영상 완성

Output: 30초 브랜드 필름
